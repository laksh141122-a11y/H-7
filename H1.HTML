CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in a markup language, most commonly HTML


It is a fundamental technology of the World Wide Web, working alongside HTML and JavaScript to create interactive and visually appealing web pages.


CSS controls the visual appearance of web content, including properties like fonts, colors, spacing, layout (e.g., multi-column layouts), and positioning of elements.


A core principle of CSS is the separation of content (HTML) from presentation (CSS). This makes websites easier to maintain, modify, and adapt to different devices and media types.
